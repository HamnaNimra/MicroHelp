# MicroHelp

Neighborly micro-help platform – give or get help from your community.

## Tech Stack

- **Flutter** – Cross-platform UI
- **Firebase** – Auth, Firestore, Cloud Messaging, Cloud Functions

## Setup

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install)
- [Firebase CLI](https://firebase.google.com/docs/cli) (`npm install -g firebase-tools`)

### 1. Generate Flutter platform folders

If `android/`, `ios/`, and `web/` are missing, run:

```bash
flutter create . --org com.microhelp
```

### 2. Install dependencies

```bash
flutter pub get
```

### 3. Configure Firebase

1. Create a project at [Firebase Console](https://console.firebase.google.com).
2. Install FlutterFire CLI and configure:

```bash
dart pub global activate flutterfire_cli
flutterfire configure
```

3. Deploy Firestore rules and indexes:

```bash
firebase deploy --only firestore
```

### 4. Run the app

```bash
flutter run
```

## Project Structure

```
lib/
├── main.dart           # Entry point, Firebase init
├── app.dart            # MaterialApp, routing
├── firebase_options.dart  # Generated by flutterfire configure
├── models/             # UserModel, PostModel, MessageModel
├── services/           # AuthService, FirestoreService
├── screens/            # Splash, Landing, Auth, etc.
└── widgets/            # Reusable components
```
